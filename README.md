# Desafios em JavaScript: Do Iniciante ao Avançado

Este repositório contém uma coleção de desafios em JavaScript, projetados para aprimorar suas habilidades de programação, desde conceitos básicos até tópicos avançados.

---

## Conteúdos

1. **Introdução ao JavaScript**
   - Sintaxe básica
   - Declaração de variáveis
   - Tipos de dados

2. **Estruturas de Controle**
   - Condicionais (`if`, `else`, `switch`)
   - Laços de repetição (`for`, `while`, `do...while`)

3. **Funções**
   - Declaração e invocação
   - Funções anônimas e arrow functions
   - Parâmetros e retorno de funções

4. **Manipulação de Arrays e Objetos**
   - Métodos de arrays (`map`, `filter`, `reduce`)
   - Manipulação de objetos

5. **Programação Assíncrona**
   - Promises
   - Async/Await
   - Manipulação de eventos

6. **Manipulação do DOM**
   - Seleção de elementos
   - Eventos do DOM
   - Alteração de estilos e conteúdo

7. **Trabalhando com APIs**
   - Requisições HTTP com `fetch`
   - Consumo de APIs RESTful

8. **Testes Automatizados**
   - Introdução a testes
   - Ferramentas de teste em JavaScript

---

## Contribuições

Contribuições são bem-vindas! Sinta-se à vontade para abrir issues para relatar problemas ou sugerir melhorias. Se desejar contribuir com soluções ou novos desafios, faça um fork deste repositório, crie uma branch com suas alterações e envie um pull request.
